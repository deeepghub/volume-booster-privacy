# 🔊 Volume Booster

Boost audio playback up to **600%** directly inside your browser tabs — with **true per‑tab volume control** and flawless fullscreen support for YouTube, Twitch, and more.

## ✨ Features
- 📌 **Per‑tab volume control** — each tab can have its own independent volume setting.
- 🎵 **Boost up to 600%** — go beyond system limits for quiet videos or streams.
- 🖥️ **Fullscreen that works** — reliable fullscreen playback without breaking YouTube or Twitch.
- ⚡ **Quick presets** — one‑click buttons for common boost levels (150%, 300%, 600%).
- 💾 **Local preferences** — default volume levels saved in your browser, no external storage.

## 🔒 Privacy
- This extension is **completely independent** — no servers, no databases, no external connections.
- **No data is collected, stored, or transmitted.**  
- All functionality runs locally in your browser.

## 📦 Installation
1. Clone this repo:
   ```bash
   git clone https://github.com/deeepghub/volume-booster.git
2. Open Chrome → go to chrome://extensions/.

3. Enable Developer mode (top right).

4. Click Load unpacked and select the project folder.

5. The extension will appear in your toolbar.
